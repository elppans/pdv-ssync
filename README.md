# pdv-ssync
pdv ssh sync  
- Dependências:  

>rsync  
ssh  
sshpass  
openssh  
openssl, 1.1.0+   

- Instalando

1) Deve baixar o pacote via git em `/opt`;
2) Deve executar `pdv-link.sh` para fixar os links no sistema.

- Comandos Básicos:
```
pdv-status, pdv-release (opcional), pdv-update-ssh.
```
- Sincronizar telas:
```
pdv-descanso
```
- FULL:
```
pdv-upgrade
```
